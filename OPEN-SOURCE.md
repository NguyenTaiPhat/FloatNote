# Open Source Information

## FloatNote - Free & Open Source

FloatNote là dự án **mã nguồn mở hoàn toàn miễn phí** (MIT License).

## Mục đích

- **Phục vụ cộng đồng**: Mọi người có thể dùng, học hỏi, và đóng góp
- **Tự do sử dụng**: Build và chạy cho nhu cầu cá nhân
- **Cải tiến chung**: Đóng góp code, báo lỗi, đề xuất tính năng

## Sử dụng

### Option 1: Download Release (Recommended)

1. Vào [Releases](https://github.com/YOUR_USERNAME/floatnote/releases)
2. Download `FloatNote-Setup-x.x.x.exe`
3. Cài đặt và sử dụng

### Option 2: Build từ Source

```bash
# Clone repo
git clone https://github.com/YOUR_USERNAME/floatnote.git
cd floatnote

# Install dependencies
npm install

# Development
npm run dev

# Build
npm run build:win
```

## Auto-Update

**Lưu ý:** Builds public từ repo này **không có auto-update**.

### Tại sao?

Auto-update yêu cầu:
- Server/repo để host updates
- Infrastructure để distribute
- Maintenance ongoing

### Giải pháp:

Users có thể:
1. **Manual update**: Download installer mới khi có version mới
2. **Build riêng**: Fork repo và setup auto-update cho riêng mình
3. **Theo dõi releases**: Watch repo để nhận notification
