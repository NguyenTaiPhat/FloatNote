# Changelog

All notable changes to FloatNote will be documented in this file.

## [1.0.0] - 2024-01-15

### Added
- ✨ Initial release
- 🎴 Smart Cards system with 6+ card types
- 🔍 Global search with fuzzy matching
- ⌨️ Command palette
- 🏢 Workspace management
- 🎨 Graph view for card relationships
- 📊 Timeline view
- 🎮 6 widget types (Todo, Clock, Weather, Countdown, SystemMonitor, Quote)
- 📌 Edge dock with quick access
- 👻 Boss mode (hide all windows)
- 🌐 Auto URL detection and metadata extraction
- 🤖 AI-powered tag suggestions
- 💾 SQLite database with WAL mode
- 🎯 60 FPS smooth animations
- 🎨 Beautiful dark theme with glass morphism
- 📤 Export to JSON/Markdown
- 📥 Import from JSON
- ⚡ Performance optimized
- 📱 Responsive design
- ⌨️ Global keyboard shortcuts
